## Journey 1: Airport Traveler (MIA)

**Scenario:** A traveler flying into Miami International Airport needs a rental vehicle for a 5-day trip. They want airport pickup and return.

| Stage | Detail |
|---|---|
| **Entry** | Google: "Miami airport car rental" → `/locations/miami-international-airport/` OR Home page → "Airport Delivery" section |
| **Primary Need** | Reliable vehicle ready at MIA upon arrival — no shuttle, no counter line |
| **Likely Concern** | "What if my flight is delayed? How do I actually get the car?" |
| **Required Info** | MIA pickup procedure, flight-tracking policy, vehicle options with luggage capacity |
| **Main CTA** | "View Vehicles" → filtered to airport-suitable vehicles, then "Check Availability" |
| **Trust Signal** | Real vehicle photos, clear MIA pickup instructions, Turo reviews mentioning airport delivery |
| **Policy Dependency** | Flight-delay policy, after-hours pickup, return procedure |
| **Support Path** | WhatsApp/call for flight changes, `/support/` for common questions |
| **Conversion** | Select vehicle → check dates → (future: complete booking in Wheelbase) |
| **Post-Booking** | Confirmation with exact MIA pickup instructions, vehicle photo, contact number |

## Cross-Journey Patterns

### Common Entry Pages (by traffic source)

| Source | Likely Landing Page |
|---|---|
| Google: "Miami car rental" | Home or `/locations/miami-international-airport/` |
| Google: "car rental near me" | Home |
| Google: "[make model] rental Miami" | `/vehicles/[slug]/` |
| Google: "monthly car rental Florida" | `/rental-options/monthly/` |
| Google (Spanish): "alquiler de autos" | `/es/` |
| Turo → Direct | Home (branded search) |
| Direct type-in | Home |

### Common Trust Signals Needed Across All Journeys

1. Real vehicle photography (not stock images)
2. Turo reviews (leveraging existing platform credibility)
3. Clear, upfront policies — not hidden in fine print
4. Real local operator — photos, name, South Florida presence
5. Contact accessibility — visible phone/WhatsApp on every page
6. Professional website design (design = trust for first-time visitors)
7. No bait-and-switch — the vehicle shown is the vehicle rented

### Common Policy Dependencies

Every journey touches one or more of:
- Driver requirements (age, license)
- Insurance requirements
- Payment/deposit terms
- Cancellation policy
- Mileage policy
- Pickup/return procedures

These must be written clearly and linked consistently across all relevant pages.
